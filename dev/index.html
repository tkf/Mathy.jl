    Mathy.@$ — Macro.
    @$      op dot_expr
@$ init op dot_expr
@$      op { dot_expr | filter_expr }
@$ init op { dot_expr | filter_expr }

    Examples

    julia> using Mathy

julia> @$ + (1:10) .^ 2
385

julia> ans == sum((1:10) .^ 2)
true

julia> @$ 0.0 + (1:10) .^ 2
385.0

julia> @$ 1000 + (1:10) .^ 2
1385

julia> @$ + { (1:10) .^ 2 | isodd(_) }
165

julia> ans == sum(x for x in (1:10) .^ 2 if isodd(x))
true

julia> @$ + { (1:10) .^ 2 | (_ + 2) % 3 == 0 }
259

julia> ans == sum(x for x in (1:10) .^ 2 if (x + 2) % 3 == 0)
true

julia> ∑ = +
       ∏ = *;

julia> @$ ∑ { (1:10) .^ 2 | (_ + 2) % 3 == 0 }
259

julia> @$ ∏ { (1:10) .^ 2 | (_ + 2) % 3 == 0 }
501760000

    Curly braces can be nested (although it is not optimized yet as of Transducers 0.2.1):

    julia> @$ + { 1:10 | isodd(_) } .^ 2
165

julia> @$ 0 + { 2 .* ({ 1:10 | isodd(_) } .^ 2 .+ 1) .- 1 | _ % 3 == 0 }
153

julia> ans == sum(filter(x -> x % 3 == 0, 2 .* (filter(isodd, 1:10) .^ 2 .+ 1) .- 1))
true

julia> @$ + isodd.({ 1:10 | _ % 3 == 0 })
2

julia> @$ + isodd.{ 1:10 | _ % 3 == 0 }  # () can be omitted
2
